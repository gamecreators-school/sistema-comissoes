📋 Instruções de Setup Automático

  1. Acesse Google Apps Script
  2. Crie um novo projeto
  3. Cole o código de setup automático fornecido
  4. Mude a API_KEY na linha 8 do código
  5. Execute a função createGameCreatorsDatabase()
  6. Copie o ID da planilha gerado
  7. Faça deploy como Web App (Execute as: Me, Access: Anyone)
  8. Use o ID aqui no sistema
